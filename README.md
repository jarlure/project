目前已经通过jitpack。使用gradle的小伙伴们可以通过如下配置获取该UI框架库
 
   ```gradle
   repositories { 
        jcenter()
        maven { url "https://jitpack.io" }
   }
   dependencies {
         compile 'com.github.jarlure:project:1.0.0'
   }
   ```
