这是一个基于JME3和同系列UI框架的通用项目框架。

目前已经通过jitpack。使用gradle的小伙伴们可以配置如下：
 
   ```gradle
   repositories { 
        jcenter()
        maven { url "https://jitpack.io" }
   }
   dependencies {
         compile 'com.github.jarlure:project:1.0.1'
   }
   ```
